# Google Analytics 4 (GA4) for Google Tag Manager


Google Analytics Enhanced Ecommerce enables your magento2 store to send product impression, product purchases, check out steps and other sales data to be sent to your Google Analytics accounts.

>:warning:  Google Analytics 4 (GA4) 1.6.0 has some breaking changes to Google Tag Manager Extension. Please download the latest version of Google Tag Manger at https://github.com/magepal/magento2-google-tag-manager.
